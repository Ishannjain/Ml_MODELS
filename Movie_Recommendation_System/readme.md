# 📖 Project Overview

The Movie Recommendation System recommends movies similar to a selected movie using **Content-Based Filtering**.

Instead of recommending popular movies, the system analyzes movie content such as:

- Genres
- Overview
- Keywords
- Cast
- Director

and recommends movies having similar characteristics.

---

# 🎯 Objectives

- Build an intelligent recommendation engine.
- Learn Content-Based Filtering.
- Perform Feature Engineering.
- Apply NLP techniques.
- Calculate Cosine Similarity.
- Deploy using Streamlit.
