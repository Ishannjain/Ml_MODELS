# Design System for House Price Prediction App

## 1. Purpose
This design guide defines the visual style, color palette, typography, and layout rules for the House Price Prediction web app.

## 2. Design Style
- Style: clean, modern, professional, and data-focused
- Goal: make the app feel trustworthy, simple, and easy to use
- Tone: friendly and reliable

## 3. Color Theme

### Primary Colors
- Primary Blue: `#2563EB`
- Secondary Teal: `#14B8A6`

### Neutral Colors
- Background: `#F8FAFC`
- Surface/Card: `#FFFFFF`
- Text Primary: `#0F172A`
- Text Secondary: `#64748B`

### Accent Colors
- Success: `#16A34A`
- Warning: `#F59E0B`
- Danger: `#DC2626`

## 4. Theme Guidelines
- Use a light theme by default
- Keep the UI minimal and uncluttered
- Use soft shadows and rounded cards for a polished look
- Highlight important results such as predicted price with strong contrast

## 5. Fonts
- Primary Font: `Inter`
- Secondary Font: `Poppins`
- Fallback: `Segoe UI, Arial, sans-serif`

## 6. Typography System (4 Styles)

### 1. Hero / Page Title
- Font Size: 28-32px
- Font Weight: 700
- Line Height: 1.2

### 2. Section Heading
- Font Size: 20-24px
- Font Weight: 600
- Line Height: 1.3

### 3. Body Text
- Font Size: 15-16px
- Font Weight: 400-500
- Line Height: 1.5

### 4. Caption / Label
- Font Size: 12-13px
- Font Weight: 500
- Line Height: 1.4

## 7. UI Components
- Buttons: rounded corners, blue primary style
- Cards: white background, subtle shadow, rounded corners
- Sidebar: light neutral background for form inputs
- Result Box: prominent display for predicted value

## 8. Layout Rules
- Use a simple top-to-bottom structure
- Keep form fields grouped logically
- Place prediction results in the center or top-right of the page
- Maintain consistent spacing using an 8px grid system

## 9. Visual Example
- Header: “House Price Prediction”
- Sidebar: input fields for bedrooms, area, location, etc.
- Main Area: prediction result card and charts
