# Customer Churn Prediction

## Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a telecom customer is likely to leave the service based on demographic and service-related information.

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Multiple ML Algorithms
- Model Evaluation
- Streamlit Web Application
- Deployment Ready
